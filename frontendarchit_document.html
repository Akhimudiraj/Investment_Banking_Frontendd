The Investment Banking Portal frontend is developed using Angular 17 with Angular Material. It is a single-page application (SPA) that interacts with a Spring Boot backend via RESTful APIs, secured with JWT authentication. Key Features: User Authentication (Login / Logout) Role-based access (User / Admin) CRUD operations for deals Notes management for deals User management for Admins Search and filter functionality
